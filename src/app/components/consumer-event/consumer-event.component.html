Consumer Event

Consumer events are directed bottom up. It starts from the result observable and ends at the operator context, or the observable itself. Use [S] If you subscribe And [U] If you unsubscribe The position of the box is 0.15em above the observable. If the box has to be placed next to it on the left side and 0.15em inside the start of the observable. Other components in the same frame are queued in the next free block to the right.
