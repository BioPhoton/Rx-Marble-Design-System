Legend

The legend section is the second possible section in a diagram. It can have variable height. This section is optional too, but compared the the description section it is always part of the diagrams as we want to have the legend as part of the diagram itself. Use small font size for the legend. For font weight use normal for description and bold for indexes, operator params, or content references.

  • Time per frame
  • Name and color
  • Datatype and shape
  • Content
  • Indexe
  • Operator params
